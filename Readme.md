# Api Minecraft RCON

Minecraft RCON API to send raw minecraft nvms commandes via http requests.

## Quickstart

```sh
# Installer dependencies
npm ci

# Start application
npm run dev
```

## [POST] `/rcon`

Send RAW Minecraft nvms command lines using body as `JSON`.

**Request**

```json
{
  "command": "say hello"
}
```

**Response**

```json
{
  "message": "hello"
}
```
